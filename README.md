# Simple Interest Calculator

A simple, user-friendly application designed to calculate the interest accrued on a loan or investment over a specific time period.

## 🧮 Formula Used

The application utilizes the standard simple interest formula:

**A = P * (1 + r * t)**

Where:
- **A** = Total Accumulated Amount (Principal + Interest)
- **P** = Principal Amount (Initial sum of money)
- **r** = Annual Interest Rate (as a decimal, e.g., 5% = 0.05)
- **t** = Time Period (in years)

---

## ✨ Features

- **Instant Calculation**: Computes total interest and final balance dynamically.
- **Input Validation**: Ensures users enter valid positive numbers for all parameters.
- **Responsive Design**: Clean and fully optimized user interface for mobile and desktop screens.
